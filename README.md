# Python Api - Flask

## Docker

> Build Image

``` bash
docker build -t python-flask .
```

> Run the image

``` bash
docker run -p 3000:3000 -d python-flask
```

> See images

``` bash
docker ps
```

> Stop the image

``` bash
docker stop 9e30dc
```

> Remove/delete the image

``` bash
docker container rm 9e30dc
```
